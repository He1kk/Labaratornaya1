Отчет по лаботаротоной работе №1. Gradle. Базовое приложение Spring

Цель работы: Освоение практических навыков развертывания и настройки автоматизированного сборщика проектов Gradle,
конфигурирования зависимостей Spring Framework, а также проектирования кросс-платформенного консольного Java-приложения
со слабо связанной архитектурой для чтения и парсинга данных из внешних CSV-ресурсов.

Выполнение работы:
1) Скачал и установил Gradle
2) Добавил bin в переменную среду для обеспечения глобального доступа из командной строки.
3) Создал проект на Gradle и JDK: Axiom JDK 21.0.10.
4) Подключил Spring Framework в файле конфигурации build.gradle
5) Создал структуру проекта:
   1. Main - точка входа в приложение
   2. AppConfig - класс конфигурации Spring
   3. MessageProvider - интерфейс для получения данных
   4. MessageRenderer - интерфейс для вывода данных
   5. CarServiceMessageProvider - класс который отвечает за чтение и парсинг CSV-ресурса
   6. OutMessageRenderer - класс реализующий вывод интерфейса в консоль
6) В каталоге resources был создан файл resource.csv, в нем хранятся данные о товарах.

Выводы: В ходе лабораторной работы были успешно освоены процессы установки локального окружения Gradle 9.5.0 и интеграции его с переменными среды ОС.
Реализовано базовое приложение на платформе Java 21 и Spring Framework, демонстрирующее принципы инверсии управления при связывании компонентов через интерфейсы.
Усвоены практические навыки потокового чтения данных через InputStream и построчной обработки табличных файлов формата CSV.
